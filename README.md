Jiao is a cross platform 2D painting engine.

- [Documentation](https://docs.rs/jiao)
- [Release notes](https://github.com/RustVis/jiao/releases)

Currently these bindings are supported:
- cairo (and gtk)
- skia (via rust-skia crate)
- Qt
- web canvas (wasm)
